# Асинхронный сервер на linux

Сервер на linux с асинхронной обработкой клиентских подключений по протоколам tcp и udp с помощью механизма мультиплексирования на базе epoll 

## Установка

```bash
sudo dpkg -i server.deb
sudo systemctl start my-server
```

## Команды
- /time - возврат текущего времени и даты
- /stats - возврат статистики (общее количество подключившихся клиентов и подключенных в данный момент)
- /shutdown - завершение работы
